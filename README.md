# Auth & User Management API

API REST desenvolvida em **Java com Spring Boot** para **autenticação, autorização e gerenciamento de usuários**, aplicando boas práticas de back-end, segurança e testes automatizados.  
O projeto utiliza **Spring Security, JWT e OAuth2**, segue **arquitetura em camadas**, possui **documentação com Swagger**, **containerização com Docker** e **deploy introdutório na AWS (EC2/RDS)**.

---

##  Tecnologias Utilizadas

- Java  
- Spring Boot  
- Spring Data JPA  
- Spring Security  
- JWT / OAuth2  
- OpenFeign  
- Swagger / OpenAPI  
- JUnit 5   
- Docker  
- AWS (EC2 / RDS – nível introdutório)  
- Maven  
- Git & GitHub  

---

##  Funcionalidades

- Cadastro e gerenciamento de usuários  
- Autenticação e autorização com JWT  
- Controle de acesso por perfis (roles)  
- CRUD completo seguindo boas práticas  
- Integração com API externa (**ViaCEP**)  
- Documentação interativa com Swagger  
- Testes unitários na camada de serviço  
- Aplicação containerizada com Docker
