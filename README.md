# auth-user-management-api
API REST desenvolvida em Java com Spring Boot para autenticação, autorização e gerenciamento de usuários, utilizando Spring Security, JWT e OAuth2. Implementa CRUD completo, arquitetura em camadas, testes unitários com JUnit e Mockito, integração com API externa (ViaCEP), documentação com Swagger, containerização com Docker e deploy introdutório na AWS (EC2/RDS).
